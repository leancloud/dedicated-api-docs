/indices/{className}

Create new index for specified class.

Secured by appMasterKey

通过应用的 MasterKey 鉴权

  • Request
  • Response
  • Security

URI Parameters

  • className: required(string)

Body

Media type: application/json

Type: object

Properties
  • keys: required(object)

    Example:

    {
  "item": 1,
  "quantity": -1
}
  • unique: (boolean)

    indicates whether the index is unique or not.

    Example:

    true
  • sparse: (boolean)

    indicates whether the index is sparse or not.

    Example:

    true
  • indexName: (string)

    index name.

    Example:

    query for inventory

Example:

{
  "keys": {"pubTimestamp": 1, "pubUser": 1},
  "unique": false,
  "sparse": true,
  "indexName": "query for user"
}
